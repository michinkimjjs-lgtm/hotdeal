# HotDealMoa (핫딜모아)

국내 인기 핫딜 커뮤니티(뽐뿌, 펨코, 루리웹)의 정보를 한곳에서 모아보는 서비스입니다.

## ✨ 기능
- **실시간 크롤링**: 10분마다 최신 핫딜 정보를 자동으로 수집합니다.
- **통합 보기**: 여러 사이트의 핫딜을 한 페이지에서 편리하게 볼 수 있습니다.
- **필터링**: 카테고리별(PC, 가전, 먹거리, 패션 등)로 원하는 상품만 골라볼 수 있습니다.
- **북마크**: 관심 있는 핫딜을 저장하고 나중에 다시 볼 수 있습니다.
- **키워드 알림**: (개발 예정) 원하는 키워드가 등록되면 알림을 받을 수 있습니다.

## 🛠 기술 스택
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend (Serverless)**: Supabase (PostgreSQL, Auth, Storage)
- **Crawler**: Python, Playwright, GitHub Actions
- **Deploy**: GitHub Pages

## 🚀 자동화 상태
- **GitHub Actions**: `Every 10 minutes` (Active ✅)
